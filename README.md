## Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Expo CLI

### Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/Ingrdnt.git
```

2. Navigate to the project directory:
```bash
cd Ingrdnt
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npx expo start
```
